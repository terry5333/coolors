# Coolors Lite

A Coolors-inspired color palette generator with lock/unlock, one-click copy, and Firebase cloud saving per user.

## Features

- Generate palettes with **Space**
- **Lock** colors so they wonâ€™t change on regenerate
- Click HEX to **copy** a single color
- One-click copy the **whole palette** (top bar)
- Add more columns with **+**
- Save palettes to **Firestore** per user
- Browse saved palettes in a **Coolors-like drawer**
- Google Sign-in via **Firebase Auth**

## Tech Stack

- Next.js (App Router)
- Firebase Auth (Google provider)
- Firestore (per-user palettes)
- TypeScript
